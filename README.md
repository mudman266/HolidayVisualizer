Holiday Visualizer is a web app that puts your days off into perspective. Not the events of the day but the days themselves.

Features:
Token system that represents days off
  - Tokens can be divided in half
  - Set number of total holiday hours per year
  - Set number of hours per token
Calendar with multiple views
  - Year view shows all months
    - Each month shows each day
    - Days with a holiday token are colored
  - Month view shows each day in a larger view
    - Each event is listed within its day
Report area
  - Total hours
  - Hours (Days) spent
  - Hours (Days) remaining
